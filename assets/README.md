# Recursos Visuais (Assets)

Esta pasta conterá todos os recursos gráficos, materiais de design e capturas de tela relacionados ao projeto **Flavos OS 2.0**.

---

## 🎨 O que armazenar aqui

- **Logotipos:** Logomarcas do Flavos OS (em formatos SVG e PNG).
- **Screenshots:** Capturas de tela da interface gráfica do Web Console.
- **Mockups:** Modelos de design de interface produzidos durante a Fase 0.
- **Diagramas de Fluxo:** Arquivos PNG/SVG exportados de fluxos de dados ou diagramas de infraestrutura adicionais.

---

## 📌 Organização
Os arquivos visuais devem ser organizados em subpastas conforme crescerem:
- `assets/logos/`
- `assets/screenshots/`
- `assets/diagrams/`
